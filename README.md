# Maincraft-technology-internship-task-1-
📒 Internship-Task-1

A simple Notes Application built using Kotlin and Jetpack Compose.

---

🎯 Objective

The objective of this project is to create a single-screen Notes App using Jetpack Compose and understand the fundamentals of Android development, including UI design and state management.

---

🛠️ Tools Used

- Android Studio
- Kotlin
- Jetpack Compose
- ViewModel
- Material 3

---

⚙️ Implementation of Logic

- Created a "Note" data class to represent note objects.
- Implemented "NotesViewModel" to manage the list of notes.
- Used "Scaffold", "TopAppBar", and "FloatingActionButton" to build the UI.
- Displayed notes using "LazyColumn".
- Added functionality to create new notes through a dialog box.
- Managed application state using ViewModel.

---

🚀 Future Enhancements

- Room Database Integration
- Edit and Delete Notes
- Search Functionality
- Dark Mode Support
